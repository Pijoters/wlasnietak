Privacy policy



Introduction
imgs.world is a free private image uploader. This policy explains our policy with respect to information that is collected from visitors to our site and use of the services we provide.

Personal information we hold
Below is a list of the personal information we store at various times when you use our services.

When you regenerate your token:
  • New token
  • Your ID
When you upload a file:
  • File ID
  • Your ID
  • File content (to display)
  • File type
When you report an upload:
  • File ID
  • Your ID
  • File author ID
  • Report reason
Data deletion
If you would like to request deletion of your data, you may do so by sending a DM to pejpal#8973. Once it is determined that you are the owner of the requested account, all data will be deleted.
